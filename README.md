ATQA example
============

This is a simple project meant to demonstrate use of the Atqa framework.
See it at https://github.com/byronka/atqa

Quick start:
------------

* To run: `make run` and hit https://localhost:8080 (spoiler: it says hello to the world)
* For help: `make help`

System requirements: 
--------------------

JDK version 20 is _required_

Developed in two environments:
* MacBook Pro with OS 12.0.1, with OpenJDK 20, GNU Make 3.81 and Rsync 2.6.9
* Windows 10 64-bit professional, on Cygwin, OpenJDK 20, Gnu Make 4.4 and Rsync 3.2.7
